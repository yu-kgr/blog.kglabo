# Hugo install
```
# Homebrew
brew install hugo

```

# Hugo Command
```
# build
hugo

# Create new post
hugo new blog/ARTICLE_NAME.md

# Hugo runs its own webserver to render the files
hugo server

# theme to use (located in /themes/THEMENAME/)
hugo -t=THEMENAME

# include content marked as draft
hugo server -D

# watch filesystem for changes and recreate as needed
hugo server -w

# The most useful command
hugo server -w -D
```
